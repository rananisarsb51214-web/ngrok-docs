# ERR_NGROK_1114

### Message
Cannot use bind address '-R `HOST`:`PORT`:' with '-remote-addr=`ADDR` parameter.
You may either specify -R `HOST`:`PORT`:' or -remote-addr=`ADDR` but not both.
