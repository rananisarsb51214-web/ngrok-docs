# ERR_NGROK_201

### Message
The ngrok API requires that you use Basic or Bearer authentication via the Authorization header. It could not parse the header in this request. Authorization Header: `VAL`. API keys and instructions are available on your dashboard: https://dashboard.ngrok.com/api-keys
