# ERR_NGROK_322

### Message
This name is reserved in a different region.
Failed to create an endpoint with the domain `DOMAIN` in the region `WANT` because it is reserved
in the `HAVE` region.

Try connecting to a different region: https://ngrok.com/docs/universal-gateway/points-of-presence/
